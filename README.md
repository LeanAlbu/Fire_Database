# Projeto de Banco de Dados: Monitoramento de Queimadas no Tocantins

## Descrição do Projeto
Este projeto foi desenvolvido para a disciplina de Banco de Dados e tem como objetivo criar uma aplicação para localizar, armazenar e gerenciar dados relacionados a queimadas no estado do Tocantins. A solução utiliza **Python** para a lógica de programação e interface, e **PostgreSQL** como sistema de gerenciamento de banco de dados (SGBD) para armazenar as informações de forma estruturada.

O sistema permite:
- Registrar dados sobre queimadas (como localização, data, extensão, etc.).
- Consultar informações históricas de queimadas no Tocantins.
- Auxiliar na análise de padrões e prevenção de incêndios na região.

## Tecnologias Utilizadas
- **Python**: Linguagem principal para desenvolvimento da lógica e interface.
- **PostgreSQL**: Banco de dados relacional para armazenamento dos dados.
- **Flet**: Framework Python para criação da interface gráfica de usuário (GUI).
- **Psycopg2**: Biblioteca para integração entre Python e PostgreSQL.
